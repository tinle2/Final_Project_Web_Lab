# Tin Le Personal Website

## Overview
This is my personal portfolio website for the final project of the course.  
It showcases my background, previous work, and projects, and includes an interactive contact form.

The website is fully responsive and built using **HTML**, **CSS**, **JavaScript (ES6+)**, and **Bootstrap 5**.  
It demonstrates layout design, dynamic content rendering, user interface components, and form validation.

## Live site
https://tinle2.github.io/Final_Project_Web_Lab/

## Repository
https://github.com/tinle2/Final_Project_Web_Lab

## Features
- Single-page layout with navigation to About, Previous Work, Projects, and Contact
- Responsive hero section introducing me and highlighting key information
- About section with accordion for structured background details
- Previous Work section summarizing experience, coursework, and skills
- Projects section rendered dynamically with JavaScript, including filtering by category (web/data/systems)
- Bootstrap carousel displaying snapshots of the site's features
- Contact form with validation, status messages, and demo submission stored in `localStorage`
- Light/dark theme toggle with user preference saved across visits
- Built with modular HTML, CSS, and JavaScript files

## Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **Bootstrap 5** (CSS and JS via CDN)
- **LocalStorage** for form data and theme persistence
